== MMD モーションキーリダクションツール
サイト「VPVP」(http://www.geocities.jp/higuchuu4/index.htm)で公開されている
VOCALOID 3DPV製作用ツール「MikuMikuDance」（通称MMD）のモーションデータ(拡張子vmd)の
モーションキーを削除するツールです。
Kinectなど作成した全フレームにキーが打たれているようなモーションファイルを
編集しやすいファイルにすることを想定しています。

== 概要
線形に補間したキーと実際のキーとの差分が大きいものからキーを挿入していく方法で、
それをある一定のキーの数になるまで繰り返します。
どれぐらいキーを挿入するかは設定ファイルに基づいて決めます。
デフォルトの設定値は10%から20%です。

== 動作環境
Windows7で確認しました。XPでも動くと思います。
Rubyで作成したスクリプトをExerbを使ってWindows実行形式ファイルに変換しています。

== インストール
配布ファイルを解凍するだけです。

== ファイル
* reduce_keys.exe <- 実行形式ファイル
* config.yml <- 設定ファイル
* readme.txt <- このファイル

== 使い方
解凍した reduce_keys.exe にモーションファイルをドラッグ＆ドロップすれば、
モーションファイルに_000など連番をつけたファイルを作成します。
重い場合は数分程度かかる場合もあります。
コマンドラインから実行する場合は以下のようにすれば出力ファイルを設定できます。
reduce_keys.exe [モーションファイル] [出力ファイル]

== 設定ファイル
設定ファイルはYAML形式のテキストファイルになっています(文字コードはUTF8)
キーをどの程度残すかボーン毎に設定しています。設定しているボーン以外の
モーションデータはそのままです。
センター: 0.1 <- センターボーンを10%残す設定です。
